# Layer 7. Loop

## 목적 / Purpose

Loop Layer는 Agent OS의 Feedback Loop 계층이다.

Hermes Agent를 활용해 Agent OS가 주기적으로 점검, 기록, 보고, 개선되도록 한다.

```text
Check Dashboard
→ Read active projects
→ Read related Memory
→ Review current status
→ Identify bottlenecks
→ Update Memory
→ Update Dashboard
→ Send Slack report
→ Suggest next action
```
