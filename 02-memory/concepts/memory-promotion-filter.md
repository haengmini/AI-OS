# Memory Promotion Filter

## 한 줄 정의
원문을 무작정 저장하지 않고, **5대 필터 중 하나 이상 통과분만** wiki로 승격한다.

## 핵심
```
① 향후 반복 재사용되는가
② 다른 에이전트/미래의 내가 반드시 읽어야 하는가
③ 의사결정 근거를 추적할 필요가 있는가
④ 재시도하면 안 되는 리스크인가
⑤ 전체를 일관되게 맞춰야 하는 공통 규칙인가
```
- 하나도 아니면 `raw/` 또는 일지에만. → 세컨드 브레인은 "저장"이 아니라 **증류**가 목적.
- 저장 단계의 절제 = 검색 단계의 [[progressive-disclosure]]와 짝.
