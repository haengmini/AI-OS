# Agents Layer

## 한 줄 정의
Agent OS 7-layer architecture의 Layer 4. operator, pm, researcher, analyst, coder, reviewer, reporter, governor, auditor, curator, sentinel 같은 역할 계약 계층이다.

## 핵심
- Agent는 모델명이 아니라 책임·handoff·검증 기준을 가진 역할이다.
- Kanban board와 Situation Room은 agent work routing을 드러낸다.
- Agent는 [[models-layer]] 위에서 실행되고 [[dashboard-layer]]에 상태를 남긴다.
