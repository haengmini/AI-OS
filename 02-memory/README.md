# Layer 2. Memory

## 목적 / Purpose

Memory Layer는 Agent OS의 장기 기억 저장소이다.

## Main Tools

```text
Obsidian   → 메인 지식 저장소
Graphify   → 지식 관계 시각화
NotebookLM → 문서/PDF 기반 분석
Zotero     → 논문/특허/레퍼런스 관리
```

## Core Rule

Memory는 원문 전체를 무작정 저장하지 않는다.

```text
핵심 요약
→ 적용 가능한 규칙
→ 재사용 가능한 명령어
→ 의사결정 근거
→ 다음 액션
```
