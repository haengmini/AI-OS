# Inbox 사용법

빠르게 잡은 생각, 링크, 논문 후보, Slack에서 나온 아이디어를 임시로 두는 곳입니다.

처리 규칙:

- 프로젝트 실행 항목이면 `10-projects/` 또는 해당 프로젝트 폴더로 연결
- 장기 관리 주제이면 `20-areas/`
- 참고자료이면 `30-resources/`
- 원자적 개념이면 `40-notes/` 또는 기존 `concepts/`
- 처리 완료 후 원본은 유지하되 status를 `processed`로 바꿉니다.
