# Dashboard Design Spec

## 목적 / Purpose

Agent OS Dashboard는 Agent OS의 전체 상태를 시각적으로 판단하기 위한 Web Control Center이다.

## Main Pages

1. Overview
2. Agents
3. Projects
4. Resources
5. Memory
6. Loops
7. Slack Commands

## UI Requirements

- Korean/English hybrid UI
- Mobile responsive
- Card-based layout
- Clear status colors
- Easy to read on phone
- JSON-driven data
- Minimal manual editing

## Design Rule

Use Figma and open-source templates first. Avoid unnecessary custom UI work.
